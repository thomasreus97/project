Land occupany in the Netherlands

This website provides different visualisations of the land occupancy distribution in the Netherlands. All data used for this website is from the CBS and can be seen by clicking Data source (CBS) on the top of the page.

According to the CBS, the land occupancy can be distributed into the seven different types: traffic, built, semi-built, recreation, agricultural, forest & nature and water. More information about these types can be found by clicking Description occupancies on the top of the page.

The visualisations can be viewed by clicking Visualisations on the top of the page. This page consists of the following three visualisations:

  1. A map of the Netherlands for one chosen occupancy.
  2. A piechart with the occupancy distribution of one chosen location.
  3. A (stacked) barchart of selected occupancies for all provinces.
